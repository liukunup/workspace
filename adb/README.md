## ADBEX是什么？

* ADBEX是对adb可执行程序进行了一层Python封装，以便在Python中更方便的使用：）
* 我们建议将adb可执行程序内置命令，按照不同功能封装成不同的Python对象，以便适应不同应用场合


## ADBEX有哪些功能？

* 对adb进行封装，封装后为adb_core中的exec()
* 对adb shell进行封装，封装后为adb_core中的shell()
* 支持调试模式，可打印出封装执行时的原始命令以及执行后原始结果
